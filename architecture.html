Vertical Cross-Cutting Concerns

Security

  • UI Layer: Bearer token auth (INTERNAL_API_KEY)
  • Service Layer: make_api_key_verifier(), JWT verification, AgentContext.from_jwt()
  • Domain: HMAC-signed X-Agent-Context header, OPA fail-closed policies, row-level filters, column masking
  • Infrastructure: Redis password auth, PostgreSQL roles, loopback-only ports in dev

Logging/Monitoring

  • UI Layer: Console logging, SSE event tracking
  • Service Layer: structlog JSON logging (configure_logging), setup_telemetry()
  • Domain: OTel spans on OPA calls, cache hit/miss, MCP tool calls; metrics
  • Infrastructure: OTel Collector → Dynatrace, Grafana + Loki stack

Exception Handling

  • UI Layer: SSE error events, retry logic, graceful degradation
  • Service Layer: Circuit breaker pattern, fail-closed OPA, retry with backoff
  • Domain: ToolResultCache graceful degradation (redis=None → no-op), cached_tool error skips
  • Infrastructure: Health checks on containers, asyncpg connection pool recovery
